간단한 방명록 조회 시스템

각 페이지의 이동은 새로운 창을 띄우는 방식이 아닌, 기존창의 정보를 post 방식으로 전달받아 놓고, 다시 돌아올 때
그 정보를 통해 다시 로딩하는 방식으로 구현. 

1) guestbook/list    목록화면
전체 목록을 페이징 처리하여 조회할 수 있고, 제목/내용/작성자 항목을 선태가여 검색과 페이징 처리 가능하게 구현

2) guestbook/register  등록화면
목록화면 상단의 register 버튼을 클릭하여 새로운 글을 등록하고, 등록 처리 후 다시 목록 화면으로 이동. 

3)guestbook/read  조회 화면
목록 화면에서 특정한 글을 선택하면 자동으로 해당 글의 조회화면으로 이동. 조회화면에선 수정/삭제 가 가능하고,
해당 버튼을 통해 수정/삭제 페이지로 이동 가능

4) geustbook/modify 수정/삭제 화면
수정화면에서는 삭제과 삭제가 가능하다. 삭제 시 목록 화면으로 이동하고, 
수정하는 경우엔 다시 해당 페이지의 조회화면으로 이동하여 수정된 내용 확인 가능.
